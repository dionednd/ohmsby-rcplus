# Roman Cancel Plus+ for OHMSBY Style Characters

Adds multiple Roman Cancel types to OHMSBY Style Characters

- Red is the normal Roman Cancel OHMSBY Syle Characters have. usable when the attack connects.
- Blue is usable during neutral or idle states. slows the enemy down by 50% for 60 frames, duration stacks up with active BRCs.
- Yellow is usable during blockstun. pushes the enemy away. forward + x is just guard reject, not roman cancel.
- Purple is usable during the startup or recovery frames of an attack.
